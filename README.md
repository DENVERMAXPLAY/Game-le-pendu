# Jeu du Pendu

## Description
Jeu de stratégie en **Python** qui consiste à deviner un mot lettre par lettre.
Chaque mauvaise réponse coûte une vie. Le joueur dispose de **6 vies** avant que
le pendu soit complet. Un dessin ASCII évolue à chaque erreur.

## Prérequis
- Python 3.x

## Utilisation

**Via le terminal :**
```bash
python main.py
```

**Via PyCharm :**
Ouvre `main.py` et clique sur **Run**.

Entrez une lettre à chaque tour et essayez de deviner le mot avant de perdre vos 6 vies !

## Exemple de résultat
```
   +-------+
   |       |
   |       O
   |      -|-
   |      | |
   |
==============

Il vous reste 0 vies.
Vous avez perdu !
```
